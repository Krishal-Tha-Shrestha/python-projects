## 📂 Projects

### 🎮 Rock Paper Scissors

A simple command-line game where the user plays Rock, Paper, Scissors against the computer.

**What it does**

* Takes user input
* Computer randomly chooses rock, paper, or scissors
* Determines the winner based on the rules of the game
* Allows repeated gameplay

Rock–Paper–Scissors is commonly used as a beginner project because it teaches how to take input, generate random choices, and decide the winner using conditions. ([Real Python][1])

**File:**
`rock-paper-scissors.py`

Run it:

```
python rock-paper-scissors.py
```

---

### 🔢 Guess the Number

A number guessing game where the computer generates a random number and the player tries to guess it.

**Features**

* Random number generation
* Feedback if the guess is too high or too low
* Loop continues until the correct answer is guessed

**File:**
`guess number.py`

Run it:

```
python "guess number.py"
```
